## set_page_options ##

```python
fpdf.set_page_options(option: str, value)
```

### Description ###

Defines page option. This functions allow set rotation and some other options.

### Parameters ###

option:
> * rotate - define page rotation (0, 90, 180 and 270).

value:
> depends on option.
